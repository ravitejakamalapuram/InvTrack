# Graphics Assets

## Required Graphics

### 1. Feature Graphic (REQUIRED)
- **Size**: 1024 x 500 pixels
- **Format**: PNG or JPEG
- **File**: `feature_graphic.png`

Design tips:
- Use app icon prominently
- Include app name "InvTracker"
- Show key value prop: "Track Alternative Investments"
- Use brand colors: #6C63FF (purple), #1A1A2E (dark background)

### 2. High-Res Icon (REQUIRED)
- **Size**: 512 x 512 pixels
- **Format**: PNG (32-bit with alpha)
- **File**: `icon_512.png`

Already generated! Copy from:
`android/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png` (192x192)
Or regenerate at 512x512 from `assets/icons/app_icon.png`

### 3. Promo Graphic (OPTIONAL)
- **Size**: 180 x 120 pixels
- **Format**: PNG or JPEG
- **File**: `promo_graphic.png`

## Creating Feature Graphic

### Option 1: Canva (Free)
1. Go to canva.com
2. Create custom size: 1024x500
3. Use template or design from scratch:
   - Dark background (#1A1A2E)
   - Add app icon
   - Add "InvTracker" text
   - Add tagline: "Track Alternative Investments with XIRR"

### Option 2: Figma (Free)
1. Create new file
2. Frame: 1024x500
3. Design with brand assets

### Brand Colors
- Primary Purple: #6C63FF
- Dark Background: #1A1A2E
- Success Green: #10B981
- Error Red: #EF4444
- Text White: #FFFFFF

### Brand Fonts
- Headlines: Bold, clean sans-serif
- Body: Regular weight
