## 🚀 Features Integrated

### **1. Auto-Fix Tools** ✅
Appear as checkboxes in PR comments - check to apply fixes:

| Tool | What It Does | When to Use |
|------|--------------|-------------|
| **localization_check** | Finds hardcoded strings, suggests ARB entries | After UI changes |
| **privacy_check** | Finds financial data without PrivacyProtectionWrapper | After financial displays |
| **multi_currency_check** | Validates currency handling compliance | After currency changes |
| **accessibility_check** | Finds missing semantic labels, touch targets | After widget changes |
| **architecture_check** | Detects layer boundary violations | After refactoring |

## 🎯 How to Use

### **In Every PR:**

1. **CodeRabbit auto-reviews** your changes
2. **Check "Finishing Touches"** section for auto-fix tools
3. **Select relevant checks** (localization, privacy, etc.)
4. **Choose action:**
   - ☐ Commit on current branch (apply fixes directly)
   - ☐ Create PR (separate PR with fixes)
5. **Ask questions** with `@coderabbitai` if needed

### **Example Workflow:**

```bash
# 1. Create PR
gh pr create --title "feat: add new investment type"

# 2. CodeRabbit reviews automatically
# 3. In PR comments, check boxes:
☑ localization_check: Commit on current branch
☑ privacy_check: Commit on current branch
☑ Generate unit tests: Post copyable unit tests in comment

# 4. CodeRabbit applies fixes and posts test suggestions
# 5. Review and merge!
```
